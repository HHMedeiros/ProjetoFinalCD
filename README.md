# ProjetoFinalCD
Projeto final do curso de ciência de dados. 

  Neste foi dada uma base de dados para que usando tecnicias de machine learning conseguissemos diminuir a quantidade de inadimpelntes que era de 35% para no maximo 25%, na empresa, consegui chegar a um modelo com precisao de 90%. podendo ser utilizados para classificação de clientes.
  
 - Nesse projeto foi realizada limpeza e tratamentos dos dados, foi feita uma analise descritiva dos dados.
 - Foi feito o Balaceamento dos dados para melhor previsao do modelo.
 - Foi ultilizado 3 tipos de modelos, para comparar os resultados e escolher o melhor. 

ps: apos a criaçao do modelo, para verificar se a qualidade do oversampling foi realmente boa, foi feita a previsao de todos os dados originais, o modelo teve precisao acima de 97%.
